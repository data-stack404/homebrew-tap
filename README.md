# Data-stack404 Tap

## How do I install these formulae?

`brew install data-stack404/tap/<formula>`

Or `brew tap data-stack404/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "data-stack404/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
